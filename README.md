# 🧠 Focus AI

Focus AI is a modern, minimalist web application built to help you manage distractions, build consistency with daily habits, and get deep work done using time-tested strategies like the Pomodoro technique.

🌍 **Live Demo:** [https://focus-ai-three.vercel.app](https://focus-ai-three.vercel.app)

---

## ✨ Features

- 🛡️ **App Blocker**: Add the apps that distract you (e.g. Instagram, YouTube) and set daily time limits. Keep track of what takes away your focus.
- ⏱️ **Pomodoro Timer**: A beautiful circular animated timer that supports 25-minute focus blocks, alongside 5-minute and 15-minute breaks. Features randomized motivational quotes when you start your sessions, and drops confetti when you complete them!
- 📅 **Focus Schedule**: A recurring weekly calendar view where you can block out study or deep work sessions. Add sessions via an intuitive modal.
- ✅ **Habit Tracker**: A visual 28-day grid journey to help you track your streaks. Mark your daily goals and click on the grid directly to log your consistency.
- 📊 **AI Insights Dashboard**: Weekly analytics on your focus score, top distractions graphed via interactive charts, and actionable AI-driven analysis.

## 🛠️ Built With

- **Framework**: React + Vite
- **Styling**: Tailwind CSS (Dark theme with `#7c6af5` accent and customized drop shadows)
- **Icons**: `lucide-react`
- **Charts**: `recharts` for dynamic distraction mapping
- **Animations**: `canvas-confetti` and native Tailwind transitions

## 🚀 Getting Started

To run this project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pragati-26/Focus-AI.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd Focus-AI
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   Navigate to `http://localhost:5173/`
